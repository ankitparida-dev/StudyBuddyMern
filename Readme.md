## 📋 Overview

StudyBuddy is a comprehensive study management platform designed for JEE and NEET aspirants. It combines traditional study tools with AI-powered assistance to create a personalized learning experience.

**Live Demo:** [https://studybuddy.vercel.app](https://study-buddy-mern.vercel.app/)  
**GitHub Repo:** [https://github.com/yourusername/studybuddy](https://github.com/ankitparida-dev/StudyBuddyMern)

## ✨ Features

### 📊 Dashboard
- Progress tracking with accuracy metrics
- Study plan with weekly progress
- Subject-wise performance analytics
- Streak tracking and recent sessions

### ⏱️ Study Tools
- **Pomodoro Timer** - 25/5 minute focus/break cycles with subject selection
- **Daily Goals** - Set and track study objectives with priority levels
- **Focus Mode** - Distraction-free environment with music and overlay
- **Quick Sessions** - 15, 25, 45, 60 minute presets
- **Smart Breaks** - 20-20-20 rule, walk reminders, mindfulness exercises

### 🤖 AI Chat Assistant
- Powered by Google Gemini AI
- Context-aware responses based on your study history
- Chat history with session management
- Smart suggestions and quick questions
- Real-time typing indicators

### 🎨 User Experience
- Dark/Light mode toggle with persistence
- Fully responsive design for mobile, tablet, and desktop
- Smooth animations and transitions
- Real-time notifications
- Offline support with localStorage fallback

## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/yourusername/studybuddy.git
cd studybuddy

# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install

# Run backend (Terminal 1)
cd backend
npm run dev

# Run frontend (Terminal 2)
cd frontend
npm run dev
